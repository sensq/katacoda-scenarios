# Docker基本操作

## コンテナを起動する

hello-worldのイメージからコンテナを起動します。

```bash
docker run hello-world
```{{execute}}

## コンテナリストを確認する

```bash
docker container ls -a
```{{execute}}


## イメージリストを確認する

```bash
docker image ls -a
```{{execute}}
